# 🚗 CarRental - Online Car Booking Platform

CarRental is a **modern full-stack web application** that allows users to browse, book, and manage rental cars effortlessly. The platform is designed with a focus on **clean UI, smooth user experience, and scalable backend architecture**.

---

## 🌐 Live Demo

🚀 **Live App** → https://car-rental-iota-sandy.vercel.app/

---

## 🚀 Features

### 👤 User
- Secure authentication (Signup / Login)
- Browse cars with detailed information
- Filter cars by category, price, and availability
- Seamless car booking experience
- View and manage bookings

### 🛠 Admin
- Add, update, and delete car listings
- Manage bookings and users
- Monitor platform activity
